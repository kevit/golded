If you've found an error in the golded+, please fill the
following form and post report by one of these ways:
- post into fidonet international echoconference GOLDED or
  russian echo RU.GOLDED;
- post into bugtrack page of the golded-plus project in the
  sourceforge cite (http://sourceforge.net/projects/golded-plus,
  the link "bug");
- send report to the maintainer via email or netmail.
